# Bandung Times
Dummy news post app

## Build
- Json: https://jsonplaceholder.typicode.com/
- Xcode 12+
- Swift 5

## Installation
Before running the project, need to type this command in the terminal after clone the project:

```sh
$ pod install
```
After this, go to Xcode > Open > select your xcode project with .xcworkspace extension.

Note: If you don't have cocoapods installed on your MacOS, can refer to [here](https://guides.cocoapods.org/using/getting-started.html).
